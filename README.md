# NodeInHeroku
First sample node application setup to run in Heroku Cloud hosting

Steps to run this sample Node application:
------------------------------------------

Clone git:
$ git clone https://github.com/gokulnathpalanivel/NodeInHeroku.git
$ cd NodeInHeroku/

Create Heroku app:
$ heroku create

Deploy the code:
$ git push heroku master

Open the website:
$ heroku open
